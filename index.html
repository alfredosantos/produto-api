🚀 Comece Rápido (3 Passos)

  1. Compilar: .\gradlew.bat clean build -x test
  2. Executar: .\gradlew.bat bootRun
  3. Testar: curl http://localhost:8080/api/produtos

🔗 7 Endpoints Implementados

GET /api/produtos - Listar todos os produtos
GET /api/produtos/contar - Contar registros
GET /api/produtos/{id} - Buscar por ID
GET /api/produtos/buscar/{nome} - Buscar por nome
POST /api/produtos - Criar produto
PUT /api/produtos/{id} - Atualizar produto
DELETE /api/produtos/{id} - Deletar produto
